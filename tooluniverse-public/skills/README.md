# ToolUniverse Skills

This folder contains public copies of the ToolUniverse-specific skills used during the plugin work.

## Included Skills

- `tooluniverse-codex-plugin`
- `tooluniverse-research-intake`
- `tooluniverse-tool-discovery`
- `tooluniverse-troubleshooting`

## How To Read Them

Start with `tooluniverse-codex-plugin` if you are working on the plugin itself.

Use the other three together as a workflow:

1. decide whether the request belongs in ToolUniverse
2. search for the right tool
3. inspect if the choice is still ambiguous
4. troubleshoot only when the plugin-local path fails
